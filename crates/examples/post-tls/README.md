# Post-TLS Interactive Zero Knowledge Proof over Committments
This example demonstrates how to run an interactive zero-knowledge proof using notarized commitments from a previous zkTLS session. It shows how to prove statements about a TLS transcript after the session, without revealing the underlying data.

## Preliminaries 
This branch comes already with the examples attestations version-controlled in.
However, if you want to recreate the attestations, follow the [attestation example](../attestation/README.md).

## Run
To run the post-TLS interactive zero-knowledge proof examples, use the following commands from the root of the repository:

### Prover

This command runs the prover, which generates a proof over a previously notarized TLS transcript:

```bash
cargo run --example prove-post-tls
```

### Verifier

This command runs the verifier, which checks the proof generated by the prover against the notarized commitments:

```bash
cargo run --example verify-post-tls 
```
